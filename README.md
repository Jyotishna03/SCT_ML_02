# Hand Gesture Recognition (MediaPipe + TensorFlow + OpenCV)

A real-time hand gesture recognition system built using **MediaPipe**, **TensorFlow Lite**, **OpenCV**, and Python.  
It detects hand landmarks through the webcam, preprocesses them, and classifies gestures using custom-trained models.
